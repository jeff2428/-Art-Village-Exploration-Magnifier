# Assets — 藝素村探險放大鏡

本目錄存放 PR5 插畫童書風美化所需的靜態資產。

## 目錄結構

```
assets/
├── illustrations/
│   ├── welcome/         # 歡迎頁背景紋理（月桃葉、台灣欒樹葉、台灣山蘇）
│   ├── animal-mode/     # 模式切換插畫（台灣藍鵲、五色鳥）
│   └── gallery-empty/   # 圖鑑空狀態插畫
├── textures/            # 紙材紋理（牛皮紙、棉紙）
└── README.md            # 本文件
```

## 使用策略

目前「插畫」透過 emoji + Flet Container 漸層/styling 呈現，
待後續版本可置換為內聯 SVG 或獨立圖片檔。

- **內聯 SVG**（純手寫）— 主要方案
- **OpenMoji**（CC BY-SA 4.0）— 候選，僅用於動物插畫
- 大於 4KB 的 SVG 走外部檔（透過 `ft.Image(src=...)`）
